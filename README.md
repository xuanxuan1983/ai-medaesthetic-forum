# The Growth Matrix

> 从心到脸，确定性变美

基于 A2A 协议的医美去中心化信任增长社区

## 🎯 项目概述

**The Growth Matrix**通过六大 Agent 矩阵（心理、审美、合规、沟通、博弈、成交），消除医美行业内耗，让求美者实现"从心到脸"的确定性变美。

## 💻 技术栈

- **前端：** Next.js 14 (App Router) + Tailwind CSS + shadcn/ui
- **后端：** Next.js API Routes
- **认证：** SecondMe OAuth2
- **数据库：** Supabase (PostgreSQL)
- **AI Agent：** SecondMe A2A Platform + Claude 3.5 Sonnet
- **部署：** Vercel

---
