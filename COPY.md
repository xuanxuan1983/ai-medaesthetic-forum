## 功能介绍文案

### 六大 Agent 矩阵

#### 1. 心理 Agent

```
标题：
先问问自己：我真的准备好了吗？

描述：
医美不只是技术问题，更是心理问题。
心理 Agent 会评估你的：
• 心理准备度（是否适合做医美）
• 动机真实性（内在需求 vs 外在压力）
• 预期合理性（现实预期 vs 不切实际幻想）
• 心理风险（身体畸形障碍、过度依赖等）

输出：
📊 心理准备度评分（0-100）
⚠️ 风险等级（低/中/高）
💡 建议（继续/暂缓/心理咨询）

为什么重要：
"很多医美失败，不是技术问题，而是心理没准备好。"
```

#### 2. 审美 Agent

```
标题：
你的脸，适合什么样的变美方案？

描述：
审美 Agent 会科学分析你的面部结构：
• 骨骼结构（三庭五眼、面部比例）
• 软组织状态（皮肤、脂肪、肌肉）
• 审美评估（符合度、改善空间）
• 方案推荐（手术 vs 非手术）

输出：
📸 面部分析报告（可视化）
✨ 推荐方案（优先级排序）
🎯 预期效果（保守/理想/最佳）

为什么重要：
"不是所有人都适合同一种方案，科学分析才能找到最适合你的。"
```

#### 3. 合规 Agent

```
标题：
这家机构，真的靠谱吗？

描述：
合规 Agent 会帮你验证：
• 机构资质（营业执照、医疗许可）
• 医生背景（执业证、专业背景、案例）
• 合规性（是否超范围经营）
• 风险预警（黑名单、投诉记录）

输出：
✅ 合规评分（0-100）
🚦 风险提示（红/黄/绿灯）
🏥 推荐机构列表（按合规度排序）

为什么重要：
"医美行业乱象频发，合规是底线，不能妥协。"
```

#### 4. 沟通 Agent

```
标题：
听不懂医生说的话？让 AI 帮你翻译

描述：
沟通 Agent 会帮你：
• 翻译医学术语（医学术语 → 通俗语言）
• 解读方案（手术流程、恢复期、注意事项）
• 管理预期（可能效果、可能风险）
• 整理问题（帮你问对问题）

输出：
📝 通俗化方案说明
❓ 问题清单（咨询时必问）
📊 预期管理报告

为什么重要：
"信息不对称是医美最大的问题，沟通 Agent 让你和医生在同一频道。"
```

#### 5. 博弈 Agent

```
标题：
同样的项目，为什么价格差这么多？

描述：
博弈 Agent 会帮你：
• 分析价格（市场价、合理价、底价）
• 制定策略（何时谈、如何谈）
• 优化方案（性价比最高的组合）
• 平衡利益（你的利益 vs 机构利益）

输出：
💰 价格区间（市场价/合理价/目标价）
💬 谈判建议（话术、时机、底线）
🎯 优化方案（省钱不省效果）

为什么重要：
"医美价格水很深，博弈 Agent 帮你找到性价比最高的方案。"
```

#### 6. 成交 Agent

```
标题：
最后一步：我该做吗？

描述：
成交 Agent 会综合前五个 Agent 的评估：
• 综合评分（多维度评估）
• 风险评估（最终决策的风险等级）
• 决策建议（做/不做/换方案）
• 成交支持（合同审查、付款建议）

输出：
📄 《确定性变美报告》
✅ 最终决策建议
📋 成交清单（需要确认的事项）

为什么重要：
"决策是你的，但 AI 会给你最全面的信息支持。"
```

## 文案使用规则

### 语气和语调

**✅ 应该：**
- 温暖、亲和
- 专业、可信
- 通俗易懂
- 以用户为中心

**❌ 避免：**
- 冰冷、机械
- 过度营销
- 医学术语堆砌
- 夸大承诺

### 文案长度

**标题：** 6-12 字
**副标题：** 15-30 字
**描述：** 50-150 字
**CTA：** 4-8 字

### 文案检查清单

- [ ] 是否通俗易懂？
- [ ] 是否以用户为中心？
- [ ] 是否传达了核心价值？
- [ ] 是否有明确的 CTA？
- [ ] 是否符合品牌语气？
- [ ] 是否避免了医学术语？
- [ ] 是否真实不夸大？
