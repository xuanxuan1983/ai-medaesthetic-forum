## 核心问题与解决方案

### 医美行业痛点

1. **信息不对称** - 求美者不懂医学术语，机构过度营销
2. **决策焦虑** - 担心效果、风险、价格、医生资质
3. **信任缺失** - 行业乱象频发，求美者无从判断
4. **沟通成本高** - 需要咨询多家机构，耗时耗力
5. **价格不透明** - 同样项目价格差异巨大，不知如何选择

### The Growth Matrix 的解决方案

通过 **六大 Agent 矩阵** 建立去中心化的信任评估体系：

```
求美者 AI 分身 ←→ 六大专业 Agent 网络
                    ↓
            多维度评估 + 透明化决策
                    ↓
            《确定性变美报告》
                    ↓
            匹配 + 成交 + 持续跟踪
```

## 六大 Agent 矩阵详解

### 1. 心理 Agent（Psychology Agent）
**职责：** 评估求美者的心理准备度和动机真实性

**核心功能：**
- 心理健康评估（是否适合做医美）
- 动机分析（内在需求 vs 外在压力）
- 预期管理（现实预期 vs 不切实际幻想）
- 心理风险预警（身体畸形障碍、过度依赖等）

**输出：**
- 心理准备度评分（0-100）
- 风险等级（低/中/高）
- 建议（继续/暂缓/心理咨询）

### 2. 审美 Agent（Aesthetic Agent）
**职责：** 分析面部结构，推荐科学的变美方案

**核心功能：**
- 面部结构分析（骨骼、软组织、比例）
- 审美评估（符合度、改善空间）
- 方案推荐（手术 vs 非手术）
- 效果预测（基于数据库）

**输出：**
- 面部分析报告
- 推荐方案（优先级排序）
- 预期效果（保守/理想/最佳）

### 3. 合规 Agent（Compliance Agent）
**职责：** 验证机构资质、医生背景、合规性

**核心功能：**
- 机构资质验证（营业执照、医疗许可）
- 医生背景调查（执业证、专业背景、案例）
- 合规性检查（是否超范围经营）
- 风险预警（黑名单、投诉记录）

**输出：**
- 合规评分（0-100）
- 风险提示（红/黄/绿灯）
- 推荐机构列表（按合规度排序）

### 4. 沟通 Agent（Communication Agent）
**职责：** 翻译医学术语，管理预期，促进理解

**核心功能：**
- 术语翻译（医学术语 → 通俗语言）
- 方案解读（手术流程、恢复期、注意事项）
- 预期管理（可能效果、可能风险）
- 问题整理（帮求美者问对问题）

**输出：**
- 通俗化方案说明
- 问题清单（咨询时必问）
- 预期管理报告

### 5. 博弈 Agent（Negotiation Agent）
**职责：** 价格谈判、方案优化、利益平衡

**核心功能：**
- 价格分析（市场价、合理价、底价）
- 谈判策略（何时谈、如何谈）
- 方案优化（性价比最高的组合）
- 利益平衡（求美者 vs 机构）

**输出：**
- 价格区间（市场价/合理价/目标价）
- 谈判建议（话术、时机、底线）
- 优化方案（省钱不省效果）

### 6. 成交 Agent（Transaction Agent）
**职责：** 风险评估、决策支持、成交促进

**核心功能：**
- 综合评分（前五个 Agent 的综合评估）
- 风险评估（最终决策的风险等级）
- 决策建议（做/不做/换方案）
- 成交支持（合同审查、付款建议）

**输出：**
- 《确定性变美报告》
- 最终决策建议
- 成交清单（需要确认的事项）

## 核心交互流程

### 阶段 1：创建 AI 分身
```
用户注册 → OAuth 登录 → 创建 AI 分身
                          ↓
                    填写基础信息
                    - 年龄、性别、职业
                    - 变美诉求
                    - 预算范围
                    - 时间安排
```

### 阶段 2：六大 Agent 评估
```
AI 分身进入社区 → 六大 Agent 开始工作
                    ↓
            并行评估（24-48 小时）
            - 心理 Agent：心理评估问卷
            - 审美 Agent：上传照片分析
            - 合规 Agent：查询机构数据库
            - 沟通 Agent：整理常见问题
            - 博弈 Agent：分析价格数据
            - 成交 Agent：综合评分
```

### 阶段 3：生成报告
```
六大 Agent 完成评估 → 生成《确定性变美报告》
                        ↓
                    报告内容：
                    - 心理准备度评分
                    - 面部分析 + 推荐方案
                    - 机构/医生推荐列表
                    - 通俗化方案说明
                    - 价格区间 + 谈判策略
                    - 最终决策建议
```

### 阶段 4：匹配与成交
```
用户查看报告 → 选择机构/医生 → AI 代理沟通
                                  ↓
                            AI 完成 90% 沟通
                            - 预约咨询
                            - 询问细节
                            - 价格谈判
                                  ↓
                            人类确认最终决策
                                  ↓
                            成交 + 持续跟踪
```

## A2A 交互设计

### Agent 间的协作

```
心理 Agent ←→ 审美 Agent
    ↓              ↓
"心理准备度低" → "推荐非手术方案"

合规 Agent ←→ 博弈 Agent
    ↓              ↓
"机构资质差" → "不推荐，即使价格低"

沟通 Agent ←→ 成交 Agent
    ↓              ↓
"用户理解度低" → "延迟决策，继续沟通"
```

### Agent 间的博弈

```
审美 Agent：推荐"全脸脂肪填充"
    ↓
博弈 Agent：价格过高，推荐"分阶段填充"
    ↓
成交 Agent：综合评估，推荐"先做局部，观察效果"
```
